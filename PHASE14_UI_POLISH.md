# FargeSpace Think Tank Phase 14 UI Polish

版本：V1
日期：2026-04-10
状态：In Progress（T1401-T1408 已完成）

## 1. 目标

这一阶段只做一件事：

- 在不重做产品结构的前提下，把前台视觉精修成更安静、更像会员智库的样子

## 2. 当前视觉方向

保留方向：

1. 极简
2. 安静
3. 会员感
4. 像私人阅读桌，不像资讯门户

不改的东西：

1. 不重做后端数据结构
2. 不重做页面信息架构
3. 不因为接口变多而重做产品逻辑

## 2.1 T1401 视觉基线

- 当前状态：已完成

### 视觉主张

- 安静的会员阅读桌，克制、留白、偏编辑部，不做仪表盘感。

### 内容节奏

1. 第一屏：只讲一个主推荐
2. 第二屏：再进入最新内容与主题书架
3. 第三屏：再给合集、索引与检索入口
4. 登录相关页面：不做营销页，只做安静入口

### 交互主张

1. 只保留少量过渡动效，不做花哨反馈
2. 优先用层级、留白、边线来建立节奏，不靠堆阴影
3. 所有交互都要先服务阅读和浏览，不服务装饰

### 颜色基线

主色：

1. 墨黑：用于主标题与主要信息
2. 暖灰白：用于大面积底色
3. 石灰边线：用于结构分区

辅助色：

1. 深青灰：用于次级说明
2. 暗金米：用于少量重点提示

限制：

1. 不再同时使用蓝、紫、青三套强调色
2. 主题、合集、来源、资产的颜色差异只能保留弱区分
3. 全站只保留一套主强调色，其他都降为中性

### 字体层级

品牌层：

1. 品牌名最大
2. 首页主标题次之
3. 页面标题低于首页主标题

内容层：

1. 正文说明统一为中号
2. 元信息统一小号
3. 标签统一最小号

限制：

1. 同一屏内不超过四档字号
2. 不依赖粗体堆层级，优先依赖尺寸和留白

### 间距基线

页面级：

1. 模块之间拉开明显间距
2. 第一屏上下留白大于后续模块

模块级：

1. 标题与说明之间紧
2. 标题区与内容区之间中等
3. 内容块与内容块之间大于现在的默认间距

限制：

1. 不再出现连续小间距堆叠
2. 不在一个区域内同时出现过多小组件

### 边框与阴影基线

边框：

1. 保留细边线，作为主要结构语言
2. 圆角收敛，不再每层都大圆角

阴影：

1. 只保留弱阴影
2. 去掉卡片化的厚重投影感

限制：

1. 默认不用“每个模块一个卡片”的方式组织页面
2. 只有主推荐区可以保留更明确的视觉重心

### 标签与按钮基线

标签：

1. 主题标签最弱
2. 来源标签次之
3. 资产标签只做信息辅助，不抢标题

按钮：

1. 主按钮只有一种视觉
2. 次按钮退到文字链接或细边线按钮
3. 不在同一屏里出现多种按钮语言

## 3. 当前主要问题

1. 首页第一屏信息权重过于平均，没有唯一主角
2. 全站的边框、胶囊、浅色块太多，显得碎
3. 颜色语言不够收敛，品牌气质还没固定
4. 登录与阅读页还没有形成同一套会员站体验

## 4. 模块划分

### 模块 A：视觉系统

范围：

1. 颜色
2. 字体层级
3. 间距
4. 边框
5. 阴影
6. 按钮与标签

目标：

- 先统一视觉语法，再改页面

### 模块 B：共享壳

范围：

1. 顶部品牌区
2. 导航
3. 登录状态区
4. 页脚

目标：

- 让全站先有同一个会员站气质

### 模块 C：首页

范围：

1. 首屏主推荐
2. 最新内容
3. 主题入口
4. 合集入口
5. 搜索入口

目标：

- 把首页从“模块堆叠”改成“主推荐带路”

### 模块 D：会员入口页

范围：

1. 登录页
2. 忘记密码页
3. 重置密码页

目标：

- 让会员入口更完整，也更符合当前人工重置阶段

### 模块 E：内容阅读页

范围：

1. 内容详情页
2. 资产区
3. 来源区
4. 主题与合集区

目标：

- 强化阅读体验，降低视觉噪音

### 模块 F：浏览页

范围：

1. 主题页
2. 合集页
3. 搜索页

目标：

- 保持功能完整，同时统一视觉节奏
