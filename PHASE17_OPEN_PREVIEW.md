# FargeSpace Think Tank Phase 17 Open Preview Mode

版本：V1
日期：2026-04-13
状态：Completed

## 1. 目标

当前阶段先解决一个问题：

1. 先开放前端产品页面，方便继续精修 UI
2. 不删除会员制、登录、会话、权限代码
3. 后续恢复会员制时，只需要关闭预览开关

## 2. 原子任务

### T1701 冻结开放预览策略

当前状态：已完成

结果：

1. 开放预览是临时产品设计模式
2. 会员制不删除，只暂时绕开页面访问拦截
3. 后续要恢复会员制时，关闭 `OPEN_PREVIEW_MODE`

### T1702 新增预览开关

当前状态：已完成

结果：

1. 新增 `OPEN_PREVIEW_MODE`
2. 同时兼容 `NEXT_PUBLIC_OPEN_PREVIEW_MODE`
3. 本地开发默认开放预览
4. 生产环境默认不开放，必须显式设置开关

### T1703 放开前端页面访问

当前状态：已完成

结果：

1. 开放预览时，首页可直接打开
2. 开放预览时，搜索页可直接打开
3. 开放预览时，主题页、合集页、内容包详情页可直接打开
4. 登录页仍然保留

### T1704 弱化或隐藏后台入口

当前状态：已完成

结果：

1. 开放预览时，左侧栏不显示后台入口
2. 开放预览时，登录页不显示 Directus 后台入口
3. 登录页改为提示“先直接浏览产品页面”

### T1705 保留登录页但不强制使用

当前状态：已完成

结果：

1. 登录页仍可访问
2. 开放预览时，不会因为已有会话自动跳转
3. 登录表单仍保留，后续恢复会员制可继续使用

### T1706 检查导航跳转

当前状态：已完成

结果：

1. 左侧导航可以进入首页、主题页、合集页、内容包详情页、搜索页
2. 首页主按钮可以进入内容包详情页与搜索页
3. 登录页预览入口可以进入首页与搜索页

### T1707 做浏览器回归

当前状态：已完成

结果：

1. 已检查首页、搜索页、主题页、合集页、内容包详情页、登录页
2. 当前未发现登录拦截
3. 当前未发现错误覆盖层
4. 当前未发现横向溢出

### T1708 写回文档

当前状态：已完成

结果：

1. 本文档记录开放预览模式
2. `TODO.md` 记录 Phase 17 状态
3. `.env.example` 与 `.env.staging.example` 记录预览开关

## 3. 后续恢复会员制

恢复会员制时：

1. 设置 `OPEN_PREVIEW_MODE=false`
2. 保留原有会员登录、会话和路由保护逻辑
3. 重新执行会员登录与路由保护 QA

## 4. 后置待办

1. 排查 Vercel 上 `OPEN_PREVIEW_MODE=true` 重新部署后仍未生效的问题，重点检查变量环境、最新部署、`NEXT_PUBLIC_OPEN_PREVIEW_MODE` 兼容开关、middleware 读取情况，以及旧部署或缓存。
2. 整理管理员登录与后台入口策略。开放预览阶段前端不展示后台入口；后续恢复会员制和角色权限时，再决定后台入口是否仅管理员可见，或改为内部单独访问地址。
