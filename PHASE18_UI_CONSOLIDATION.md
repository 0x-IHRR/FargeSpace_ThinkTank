# FargeSpace Think Tank Phase 18 UI Consolidation

版本：V1
日期：2026-04-13
状态：Planned

## 1. 问题判断

当前页面不是没有结构，而是视觉层级过多：

1. 标题、标签、按钮、卡片和装饰线条都在争夺注意力
2. 首页、主题页、合集页、搜索页都带有较强封面感，但资料库页面更需要清晰的阅读和检索
3. 卡片横向信息过多，标题、摘要、来源、形式标签、日期同时出现，扫读成本偏高
4. 英文装饰标题和中文业务内容混排较多，降低统一感
5. 背景网格、斜切图形、内描边、边框密度偏高，影响内容安静度

## 2. 收敛目标

1. 保留米白、细线、会员资料库的视觉方向
2. 降低装饰存在感，让内容成为主角
3. 让每个页面只有一个主要视觉重点
4. 让列表页更像资料库工作区，而不是封面页
5. 让卡片更容易扫读，优先看标题和摘要，再看标签和操作

## 4. 执行顺序

1. T1801-T1803 先做全局收敛规则
2. T1804 先改卡片，因为卡片是混乱感的主要来源
3. T1805-T1808 按页面逐个收敛
4. T1809-T1812 做导航、移动端和最终检查

## 5. 完成标准

1. 页面仍有会员资料库气质
2. 内容比装饰更突出
3. 列表能快速扫读
4. 详情页阅读顺序清楚
5. 桌面端和移动端都不出现明显拥挤或横向溢出
